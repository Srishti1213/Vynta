🔍 Vynta - Android Forensic Suite

An AI-powered tool to detect internal data leaks and suspicious activity from Android devices using OSINT and forensic methods.

Available Interfaces

  • 🖥️ Desktop GUI (.exe Download)
  • 🌐 Web Interface (Streamlit Instructions)
  • 🛠️ Command Line Interface (CLI)

Features

  • Hidden apps detection
  • Permission audit
  • SMS & call log scanner
  • GPS timeline mapping
  • WHOIS, DNS, Port Scan, Traceroute
  • SSL certificate checks
  • Bandwidth anomaly detection
  • Social engineering detection
  • And more...
